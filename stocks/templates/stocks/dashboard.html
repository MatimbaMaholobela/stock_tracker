How Stock Tracker Works
Trading Algorithm:

The system uses a simple 3% drop rule to generate trading signals:

  1. Data Collection: Upload CSV files with ticker, date, and close price columns
  2. Signal Generation: System analyses daily price changes
  3. BUY Signal: Triggered when stock price drops 3% or more from previous day
  4. SELL Signal: Automatically triggered 5 days after a BUY signal
  5. HOLD Signal: Applied to all other days
File Format:
ticker,date,close
AAPL,2024-01-01,150.00
AAPL,2024-01-02,145.50  # -3% drop = BUY signal
